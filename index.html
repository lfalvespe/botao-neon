Botão Neon

Utilizando a pseudo-classe 'hover' para criar efeito Neon em botão

Passe o mouse sobre um botão
Azul
Verde
Vermelho
Roxo

Código CSS do Botão:



.botao {
    color: #2196f3;
    width: 160px;
    height: 50px;
    padding: 13px;
    border-radius: 10px;
    text-align: center;
    text-transform: uppercase;
    letter-spacing: 4px;
    text-decoration: none;
    font-size: 20px;
    overflow: hidden;
    transition: 0.2s;
    box-shadow: 0px 0px 10px
                rgba(255, 255, 255, 0.563);
    cursor: pointer;
}
            

.botao:hover {
    color: #255784;
    background: #2196f3;
    box-shadow: 0 0 10px #2196f3, 
                0 0 40px #2196f3,                   
                0 0 80px #2196f3;
    transition-delay: .3s;
}
